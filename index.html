Pokedex

This application is a basic design of a Pokedex written in Python using Tkinter.
You can search Pokedex, Type, and Abilities.

Pokedex

will allow you to search by Regions, Location, Name, Description, Body Style, Egg Groups, Moves, and Groups.

Regions will allow you to search for Pokemon according to its region of origin.

Location will allow you to search for Pokemon according to where they can be found in a particular region, i.e. route, road, city, town, forest, etc.
Within a particular location, you can search for pokemon based on time of day, season of the year, weather condition, or specific action such as fishing, surfing, flying, etc.
Additionally, hovering over a Pokemon will reveal the games in which they can be found in that location.

Name will allow you to search for Pokemon by the first letter of their name, Mega evolutions, different Forms i.e, gender, regional, etc, region specific forms: Alolan, Galarian, and Hisuian, and Gigantamax forms.

Description will allow you to search for Pokemon be species i.e., The Mouse Pokemon, by the first letter of the first word, then by the first word, and, if neccessary, the second word.

Body Style will allow you to search for Pokemon based on their body style:

Egg groups will allow you to search for Pokemon by breeding groups.

Moves will allow you to search for moves by Type, Dynamax moves, G-Max moves, Z-Moves, or by their first letter.

Groups will allow you to search for pokemon based on groupings such as Eeveelution which will show all of those involved in the evolution shain of Eevee or the Swords of Justice which will show all Pokemon involved with that group.

Type

will allow you to search for Pokemon by a particular type and will also allow you to see type match-up charts for a particular type.
Hovering over a value in the type chart will reveal the effectiveness of that type over another. For example, Fire is super effective against Grass/Ice.

Abilities

will allow you to search for abilities based on their first letter.

Pokemon

will show you the image, name, number, species, type, height, weight, body style, and evolution chain of the pokemon.
Additionally, it will show the Pokedex entry for any game that the pokemon was in.
Also included are buttons to switch between different forms of the pokemon i.e., gender, regional, mega evolutions, gigantamax, and shiny forms.

Wordle Aid

This application was written in Python and is an aide to use when playing Wordle.
This program reads in a list of all acceptable Wordle words and provides options for whittling the list down closer to the correct answer.
Options include Eliminate, Possible, Duplicate, Eliminate Possible, Eliminate Duplicate, and Insert.

Eliminate allows you to remove any letters you know are not in the word.
Possible allows you to add any letters that you know are in the word but do not yet know the exact position.
Duplicate allows you to insert any letter that you know has more than one appearance in the word.
Eliminate Possible allows you to remove any letters you know are in the word from positions you know they do not go.
Eliminate Duplicate allows you to clarify if you know that a known letter only appears once in the word.
Insert allows you to confirm any letter into its known final position.
Also included are the Generate and Reset options.
Generate uses all of the currently collected information to filter the list of all words down to a list that meets the current specifications.
Reset allows you to reset all specifications to a clean slate.
