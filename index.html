Pokedex

This application is a basic design of a Pokedex written in Python using Tkinter.
You can search Pokedex, Type, and Abilities.

Pokedex

will allow you to search by Regions, Location, Name, Description, Body Style, Egg Groups, Moves, and Groups.

Regions will allow you to search for Pokemon according to its region of origin.

Location will allow you to search for Pokemon according to where they can be found in a particular region, i.e. route, road, city, town, forest, etc.

Name will allow you to search for Pokemon by the first letter of their name, Mega evolutions, different Forms i.e, gender, regional, etc, region specific forms: Alolan, Galarian, and Hisuian, and Gigantamax forms.

Description will allow you to search for Pokemon be species i.e., The Mouse Pokemon, by the first letter of the first word, then by the first word, and, if neccessary, the second word.

Body Style will allow you to search for Pokemon based on their body style:

Egg groups will allow you to search for Pokemon by breeding groups.

Moves will allow you to search for moves by Type, Dynamax moves, G-Max moves, Z-Moves, or by their first letter.

Groups will allow you to search for pokemon based on groupings such as Eeveelution which will show all of those involved in the evolution shain of Eevee or the Swords of Justice which will show all Pokemon involved with that group.

Type

will allow you to search for Pokemon by a particular type and will also allow you to see type match-up charts for a particular type.
Hovering over a value in the type chart will reveal the effectiveness of that type over another. For example, Fire is super effective against Grass/Ice.

Abilities

will allow you to search for abilities based on their first letter.
